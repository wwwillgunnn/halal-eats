# Halal Eats

Halal Eats is a web application that helps users discover halal food anywhere. Whether you're exploring a new city or just looking for nearby options, Halal Eats makes it easy to find restaurants, view locations on a map, and navigate to them in real time.

Access the app here: [Halal Eats!](https://halaleats.vercel.app)

## Tech Stack

- Frontend: Next.js, React, TypeScript
- Styling: Tailwind CSS, ShadCN UI
- Maps: MapCN

## Features

- 📍 Find halal restaurants near your location
- 🗺️ View places on an interactive map
- 🧭 Get directions/routes to selected locations
- ➕ Contribute by adding new halal restaurants to the platform
- 🔎 Browse and explore multiple food options easily

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/wwwillgunnn/halal-eats.git
cd halal-eats
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### 4. Open in browser

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
